This project is a modern invoice generator web application developed using React, TypeScript, Vite, and Tailwind CSS. It helps users create professional invoices quickly, easily, and for free. 
The application provides features for entering client information, adding invoice items, calculating totals, customizing brand settings, previewing the invoice in real time, and exporting the final result as a downloadable file. Invoicel is designed with a clean, responsive, and user-friendly interface, making it accessible on both desktop and mobile devices.

https://invoicel.vercel.app/
